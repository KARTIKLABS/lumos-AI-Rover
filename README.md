# Lumos AI Rover

An AI-assisted ESP32 smart rover project focused on embedded systems, automation, wireless communication, and intelligent experimentation.

---

## Project Goals

- Learn ESP32 development
- Explore robotics fundamentals
- Build wireless rover control systems
- Experiment with sensors and automation
- Understand hardware-software integration
- Explore future AI-assisted interaction concepts

---

## Planned Features

- Wi-Fi / Bluetooth control
- Sensor integration
- Obstacle detection
- Camera support (future scope)
- AI-assisted workflows
- Modular hardware architecture
